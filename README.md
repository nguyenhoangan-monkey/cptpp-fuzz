Instead of keeping track of terminal windows, you can use these simple commands:

* **One-time environment setup:** `make setup`
* **Compile and run `fuzz_sys` on 4 cores:** `make fuzz-sys`
* **Compile and run `fuzz_buffer` on 4 cores:** `make fuzz-buffer`
* **Check status across all targets/cores:** `make status`
* **Generate output.txt file:** `make view-inputs`
* **Stop everything and clean repository:** `make clean`
